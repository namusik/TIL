# OIDC 

## OpenID Connect

    OAuth 프로토콜을 기반으로 작동.

    목적이 다르다. 

    인증(Authentication)을 위해서 사용.

        다른 플랫폼을 통해서 사용자가 누구인지를 확인하기 위해서 사용.

        사용자의 개인 정보가 담긴 id token을 확보하는 목적.

    OAuth는 인가(Authorization)을 위해서 사용.

        해당 플랫폼에 저정된 사용자의 데이터에 접근하기 위해서 사용. 

        다른 API를 호출하기 위해서 access token 확보 목적.
