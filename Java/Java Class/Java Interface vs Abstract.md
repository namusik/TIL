## 일반 클래스 vs interface
- 클래스는 객체 생성을 위한 사용자 정의 유형.
  - 프로퍼티와 메서드를 가진다.

- 인터페이스 역시 클래스와 유사한 사용자 정의 유형.
  - 대신 구현체에 의해 재정의 되는 필드와 메서드를 가진다.
